# StatsBot100
A Discord bot designed for my server. Currently a work-in-progress.

# Commands
| Command | Description | Implemented |
|---|---|---|
| ping | Check the roundtime latency of the bot | ✅ |
| say | Echo a message as the bot | ✅ |
| purge | Mass delete messages | ❌ **(WIP)**|
